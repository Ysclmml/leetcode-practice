q60_第k个排列

给出集合 `[1,2,3,…,*n*]`，其所有元素共有 *n*! 种排列。

按大小顺序列出所有排列情况，并一一标记，当 *n* = 3 时, 所有排列如下：

+ `"123"`
+ `"132"`
+ `"213"`
+ `"231"`
+ `"312"`
+ `"321"`

给定 *n* 和 *k*，返回第 *k* 个排列。

**说明:** 

+ 给定 *n* 的范围是 [1, 9]。
+ 给定 *k* 的范围是[1,  *n*!]。

```
输入: n = 3, k = 3
输出: "213"

输入: n = 4, k = 9
输出: "2314"
```

**思路**

```
给出所有排列然后去获取当前第k个肯定不合理.
需要找到规律, 然后获取每一位.

我的思路是先确认每一位的数字, 
例如 n=4, k=9
当前备选数字a[]={1,2,3,4}
n = k - 1 = 8
sb = ''
1: n / 3! = 1
	sb[0] = a[1]=2  当前备选数字为a[] = {1,3,4}
	n = 8 - 3! = 2
2: n / 2! = 1
sb[1] = a[0] = 3 sb="23" 当前备选数字为{1,4}
n = 2 - 2!= 0
3: n / 1! = 0
sb[2] = a[0]=1 sb="231" 当前备选数字为{4}
4: sb = "2314" ---> 即为最终答案
```
