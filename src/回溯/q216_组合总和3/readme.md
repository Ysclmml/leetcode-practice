q216_组合总和3

找出所有相加之和为 ***n*** 的 ***k\*** 个数的组合***。\***组合中只允许含有 1 - 9 的正整数，并且每种组合中不存在重复的数字。

**说明:**

+ 所有数字都是正整数。
+ 解集不能包含重复的组合。 

```
输入: k = 3, n = 7
输出: [[1,2,4]]

输入: k = 3, n = 9
输出: [[1,2,6], [1,3,5], [2,3,4]]
```

**思路**

```
思路与 `组合总和II` 一样, 只是条件更加简单, 一个数字只能使用一次, 这次也不需要做重复数字的处理, 比II更简单

solve(target, cur, index, tmp, reslut) 递归时 index+1即可

```
