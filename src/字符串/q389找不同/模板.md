请找出在 ***t*** 中被添加的字母。

示例 1:

```
输入：s = "abcd", t = "abcde"
输出："e"
解释：'e' 是那个被添加的字母。
```

示例 2:

```
输入：s = "", t = "y"
输出："y"

输入：s = "a", t = "aa"
输出："a"

输出："a"
```
