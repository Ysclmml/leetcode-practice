q145_二叉树的后序遍历

给定一个二叉树，返回它的 *后序* 遍历。(使用非递归的方式)

**说明：**

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

**示例 1:**

```
输入: [1,null,2,3]  
   1
    \
     2
    /
   3 

输出: [3,2,1]

```



**思路**

```
用栈模拟递归栈即可. 需要考虑的时就是如何不重复进入左右分支.
```
