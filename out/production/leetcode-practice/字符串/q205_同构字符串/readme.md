q205_同构字符串


		给定两个字符串 ***s*** 和 ***t\***，判断它们是否是同构的。

如果 ***s*** 中的字符可以被替换得到 ***t\*** ，那么这两个字符串是同构的。

所有出现的字符都必须用另一个字符替换，同时保留字符的顺序。两个字符不能映射到同一个字符上，但字符可以映射自己本身。

**示例:**

```
输入: s = "egg", t = "add"
输出: true

输入: s = "foo", t = "bar"
输出: false

输入: s = "paper", t = "title"
输出: true
```

**思路**

```java
两个字符串同时遍历
同时建立映射表, 如果表中不存在, 则添加映射, 
如果存在则比较两者是否相等. 键值是第一个串的值, 值是第二个传的值
```
