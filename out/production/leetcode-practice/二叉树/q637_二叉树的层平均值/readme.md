q637_二叉树的层平均值

给定一个非空二叉树, 返回一个由每层节点平均值组成的数组。

**说明:**

+ 节点值的范围在32位有符号整数范围内。

```
输入：
    3
   / \
  9  20
    /  \
   15   7
输出：[3, 14.5, 11]
解释：
第 0 层的平均值是 3 ,  第1层是 14.5 , 第2层是 11 。因此返回 [3, 14.5, 11] 。

```

**思路**

```
题目非常简单, 单纯的按照层数遍历, 然后求值即可

按层遍历需要使用队列辅助
```
