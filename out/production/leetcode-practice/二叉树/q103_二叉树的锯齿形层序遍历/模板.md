#### q103_二叉树的锯齿形层序遍历

给定一个二叉树，返回其节点值的锯齿形层序遍历。（即先从左往右，再从右往左进行下一层遍历，以此类推，层与层之间交替进行）。

例如：
给定二叉树 [3,9,20,null,null,15,7],



**示例**

```java
  3
   / \
  9  20
    /  \
   15   7
      
返回锯齿形层序遍历如下：
     
[
  [3],
  [20,9],
  [15,7]
]      
```



**思路**: 

```java
在层序遍历的基础上锯齿状遍历即可.  需要逆序遍历的数, 采用栈的数据结构即可. 
```
