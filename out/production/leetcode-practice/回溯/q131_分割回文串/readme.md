q131_分割回文串

给定一个字符串 *s*，将 *s* 分割成一些子串，使每个子串都是回文串。

返回 *s* 所有可能的分割方案。

**示例 1:**

```
输入: "aab"
输出:
[
  ["aa","b"],
  ["a","a","b"]
]
```

**思路**

```
使用递归的思路, 将复杂的问题分割成子问题来进行处理解决. 
例如"aab"
第一次是分割成""以及整个字符串, 然后是"a"和剩余字符的解决方案,
然后是"aa" 和"b" 两种分割方案的组合就是

```
