q78_子集

给定一组**不含重复元素**的整数数组 *nums*，返回该数组所有可能的子集（幂集）。

**说明：**解集不能包含重复的子集。

**示例:**

```
输入: nums = [1,2,3]
输出:
[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
```

**思路**

```
回溯法
若原数组长度为n 输出所有长度为0,1,2...n的字数组
递归的时候只需要保证当前数组的索引在之前的索引+1即可
```
